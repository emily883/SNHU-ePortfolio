# SNHU-ePortfolio

Capstone ePortfolio for **CS-499 Computer Science Capstone** at Southern New
Hampshire University, by **Emily Cruz**.

The artifact is a single C++ application, the **Student Portal**, that is enhanced
across the three capstone categories:

- **Enhancement One — Software Design and Engineering** (Module Three):
  refactor to a Controller / Service / Repository architecture. See
  [`artifact/enhanced/`](artifact/enhanced/) and its
  [narrative](artifact/enhanced/module-three-narrative.md).
- **Enhancement Two — Algorithms and Data Structures** (Module Four): planned.
- **Enhancement Three — Databases** (Module Five): planned. The repository layer
  currently stores data in memory and is designed so a database implementation
  can be added here.

The original starter code is preserved in [`artifact/original/`](artifact/original/)
for before/after comparison.
